## 📚 论文类型定义

### 1. 综述类论文 (Survey/Review)
- **特点**：系统性梳理某领域研究进展
- **无需实验数据**
- **字数**：30,000-50,000
- **引用**：150-300 篇
- **周期**：2-4 周
- **示例**：《Deep Learning for Computer Vision: A Survey》

### 2. 理论类论文 (Theory)
- **特点**：提出新理论、新框架、新观点
- **无需实验数据**（需数学证明/逻辑推导）
- **字数**：25,000-40,000
- **周期**：3-6 周
- **示例**：《Attention Is All You Need》

### 3. 方法类论文 (Method)
- **特点**：提出新方法、新算法、新架构
- **需要实验验证**
- **字数**：20,000-35,000
- **周期**：4-8 周
- **示例**：《BERT: Pre-training of Deep Bidirectional Transformers》

### 4. 实证类论文 (Empirical)
- **特点**：基于真实实验/观测数据
- **必须真实数据**
- **字数**：20,000-30,000
- **周期**：8-16 周
- **示例**：实验科学论文

**当前策略**：以类型 1、2 为主（无需实验数据），逐步发展类型 3、4

## 📋 学术规范要求

### 1. 科学性要求
- ✅ 所有陈述需有文献支持
- ✅ 区分事实与观点
- ✅ 承认研究局限性
- ✅ 避免过度宣称

### 2. 引用规范
- ✅ 使用 BibTeX 管理引用
- ✅ 所有引用必须真实存在
- ✅ 避免引用低质量来源
- ✅ 优先引用顶会/顶刊

### 3. 学术诚信
- ❌ 绝不编造数据
- ❌ 绝不抄袭（使用查重工具）
- ❌ 绝不一稿多投
- ❌ 绝不伪造同行评审

### 4. 作者资格
- 所有作者必须对论文有实质贡献
- 通讯作者负责论文真实性
- 利益冲突需声明

## 📖 新手指南

### 第一次使用本仓库

1. **Clone 仓库**
   ```bash
   git clone git@github.com:ENDcodeworld/-Academic-paper.git
   cd -Academic-paper
   ```

2. **查看当前论文状态**
   ```bash
   cat PAPER_STATUS.md
   ```

3. **开始新论文**
   ```bash
   git checkout -b paper-00X-topic develop
   ```

4. **完成论文后**
   ```bash
   git merge paper-00X-topic main
   git push origin main
   ```

## 📄 附录：论文模板结构

### 标准论文结构（综述类）

```markdown
# 标题

## Abstract（摘要）
- 背景（2-3 句）
- 目的（1-2 句）
- 方法（2-3 句）
- 结果（3-4 句）
- 结论（1-2 句）
- 关键词（5-8 个）

## 1. Introduction（引言）
- 1.1 研究背景
- 1.2 问题定义
- 1.3 研究动机
- 1.4 主要贡献
- 1.5 论文结构

## 2. Background（背景）
- 2.1 核心概念
- 2.2 理论基础
- 2.3 相关领域

## 3. Taxonomy（分类框架）
- 3.1 分类维度
- 3.2 各类别详解
- 3.3 对比分析

## 4. Literature Review（文献综述）
- 4.1 方向 A 综述
- 4.2 方向 B 综述
- 4.3 方向 C 综述
- ...

## 5. Open Challenges（开放挑战）
- 5.1 挑战 1
- 5.2 挑战 2
- 5.3 挑战 3

## 6. Future Directions（未来方向）
- 6.1 方向 1
- 6.2 方向 2
- 6.3 方向 3

## 7. Conclusion（结论）
- 研究总结
- 主要贡献
- 局限性
- 展望

## References（参考文献）
- 150-300 篇
- BibTeX 格式
- 按字母顺序

## Appendix（附录）
- A. 补充材料
- B. 数据表格
- C. 代码链接
```
